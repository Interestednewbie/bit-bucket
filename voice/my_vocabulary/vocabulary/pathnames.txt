### 
### Vocabulary for dictating pathnames
### 


## 
## Complete pathnames:
## 

/etc/passwd:slash etc slash password
/dev/null:slash dev slash null
/dev/null:dev null


## 
## Pathname prefixes:
## 

/bin
/dev:slash dev
/etc:slash et cetera
/sbin:slash S bin
/usr:slash user
/var:slash VAR
/var:slash var
/tmp:slash temp


## 
## Pathname components:
## 

..:dot dot
..:parent directory

.gitignore:dot git ignore
Makefile
Program Files
README
Tmp:big temp
cygdrive:Cygwin drive
l1:L one
lib
proc:short procedure
src:short source
tmp:short temp
udir:U dir
